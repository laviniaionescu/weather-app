# IT School Weather App

A weather application built with vanilla JavaScript as part of my web development course at IT School. This project demonstrates working with external APIs, DOM manipulation, and responsive design.

## Features

- Search for current weather by city (București, Timișoara, Oradea)
- Display current temperature, real feel, wind speed, and weather description
- 5-day weather forecast with hourly breakdown
- Local storage for persisting the user's selected city
- Responsive design using Bootstrap and custom CSS

## Built With

- HTML5
- CSS3 (with Bootstrap 5)
- Vanilla JavaScript
- OpenWeatherMap API

## Live Demo

https://weather-app-li.netlify.app/

## What I Learned

- Fetching data from REST APIs
- Working with asynchronous JavaScript (Promises, .then())
- Manipulating the DOM dynamically
- Structuring a project with separate utility files
- Handling user input and events
- Storing data locally with localStorage

## Setup

1. Clone the repository
2. Add your OpenWeatherMap API key in `apikey.js`
3. Open `index.html` in your browser

## Note

This was a course project built for learning purposes. The code is intentionally structured to demonstrate fundamental concepts before moving on to React.
